SE UTILIZA NETBEANS IDE19, JDK 21

Se trata de hacer una aplicacion con JAVA Swing (JFrame) conectandolo con una BD MySql

Se utiliza un driver que es: 

mysql-connector-j-8.2.0-jar

Y en el pom.xml se pone esto: 

  <dependencies>
        <dependency>
            <groupId>com.mysql</groupId>
            <artifactId>mysql-connector-j</artifactId>
            <version>8.2.0</version>
        </dependency>
    </dependencies>

Ejemplo SQL:

CREATE TABLE persona (
    idpersona INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(50)
);

CREATE TABLE vivienda (
    idvivienda INT AUTO_INCREMENT PRIMARY KEY,
    direccion VARCHAR(100),
    numero int,
    esChungo boolean,
    idpersona INT,
    FOREIGN KEY (idpersona) REFERENCES persona(idpersona)
);
